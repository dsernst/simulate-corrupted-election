# simulate-corrupted-election

## Development

This is a [Next.js](https://nextjs.org/) project.

## Getting Started

### First time

1. Fork repo
2. Clone it down with `git clone`
3. Install dependencies with `npm install` or equivalent.

### Then to start the development server:

```bash
npm run dev
```
